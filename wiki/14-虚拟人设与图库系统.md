# 虚拟人设与图库系统

220 个互不相同的虚拟账号 + 200 张主题 SVG 配图，由 Python 脚本生成与校验，打包进 APK assets。

## 人设数据规模

- 220 个虚拟账号，分 11 个分片（`personas_001.json` ~ `personas_011.json`，每片 20 条）。
- `index.json` 含 `count=220`、`files` 列表、`accountIds`（220 个 UUID，与 `avatars/index.txt` 顺序一致）。
- 每个账号含 5-10 条历史推文（`daysAgo` 1-30），共约 1500+ 条历史推文。

## 人设 Schema（PersonaDto）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string (UUID) | 全局唯一 |
| displayName | string | 中文姓名；海外华人可能"English 中文名" |
| username | string | 如 `sp_dev_431` |
| avatarSeed | string | 与 username 相同（校验脚本断言） |
| bio | string | 不超过 160 字符 |
| profession | string | 20 种之一 |
| ageRange | string | `18-24`/`25-34`/`35-44`/`45+` |
| culturalBackground | string | `华北`/`华东`/`华南`/`西南`/`西北`/`海外华人` |
| worldview | string | 世界观文本 |
| values | string | 价值观文本 |
| languageStyle | string | `正式`/`口语`/`幽默`/`犀利`/`温和`/`文艺`/`直率` |
| catchphrase | string[] | 1-2 个口头禅 |
| emojiPreference | string[] | 按年龄段 1-5 个 emoji（数据内容） |
| typoRate | float | 0.0 - 0.05 |
| activeWindows | bool[24] | 下标 0-23 对应小时 |
| historicalTweets | array | 5-10 条 `{text, mediaTheme, daysAgo}`，`mediaTheme` 可 null |

## 人设固定字段与动态字段

- 固定字段（不可变）：`displayName`/`username`/`avatarSeed`/`bio`/`profession`/`ageRange`/`culturalBackground`/`worldview`/`values`/`languageStyle`/`catchphrase`/`emojiPreference`/`typoRate`/`activeWindows`/`timezone`。
- 动态字段（AI 维护）：`PersonaDynamicFieldEntity(lifeStory/workInfo/relationshipNetwork/mood)` + `AccountEntity` 上的 denormalized 摘要（`dynamicLifeStory`/`dynamicWorkInfo`/`recentMood`）。
- 动态字段由 `PersonaUpdateWorker` 周期更新（含漂移防护）。

## 矩阵抽样保证多样性（tools/persona-gen/generate.py）

- `TOTAL_PERSONAS=220`，`SHARD_SIZE=20`，`RANDOM_SEED=20240705`（可复现）。
- 20 职业：程序员/设计师/教师/医生/律师/记者/厨师/摄影师/音乐人/作家/运动员/创业者/公务员/护士/工程师/研究员/学生/退休者/艺术家/自由职业。
- 4 年龄段、6 文化背景、7 语言风格。
- `build_matrix_slots()`：20 职业 × 11 槽 = 220；`age_idx=(n+pi)%4`，`culture_idx=(n*5+pi)%6`（步长 5 与 6 互素保证文化背景均匀分布）。
- `pick_unique_triple()`：`(worldview, values, languageStyle)` 全局唯一去重。
- `gen_display_name(culture, rng)`：按文化背景生成姓名，海外华人 60% 概率英文名+中文姓氏。
- `build_active_windows(profession)`：按职业 24 槽 bool 数组。
- `gen_historical_tweets()`：5-10 条，scene+tail 组合去重，30% 概率随机 `mediaTheme`。
- `write_outputs()`：清理旧分片 + 写 11 分片 + `index.json` + `avatars/index.txt`。
- 模板池：`BIO_PROFESSION`/`BIO_AGE_SUFFIX`、`WORLDVIEW_PROFESSION`/`WORLDVIEW_AGE_MODIFIER`、`VALUES_CULTURE`/`VALUES_PROFESSION`、`TWEET_SCENES`（按职业 8 场景）、`TWEET_TAILS_BY_STYLE`（按语言风格 4 结尾）。
- 姓名池：`SURNAMES_BY_CULTURE`/`GIVEN_BY_CULTURE`/`ENGLISH_NAMES`。

```python
# 矩阵抽样核心逻辑
TOTAL_PERSONAS = 220
SHARD_SIZE = 20
RANDOM_SEED = 20240705

def build_matrix_slots():
    slots = []
    for pi in range(11):          # 11 分片
        for n in range(SHARD_SIZE):
            age_idx = (n + pi) % 4
            culture_idx = (n * 5 + pi) % 6   # 步长 5 与 6 互素
            slots.append((pi, age_idx, culture_idx))
    return slots                  # 共 220 槽
```

## 人设校验（tools/persona-gen/validate.py）

- 纯 Python 标准库。校验项：
  1. 加载所有 `personas_*.json`；
  2. 数量 >= 220（`EXPECTED_MIN_COUNT`）；
  3. `id` 唯一；
  4. `username` 唯一；
  5. `(worldview, values, languageStyle)` 三元组无完全重复；
  6. `REQUIRED_STR_FIELDS` 非空；
  7. `activeWindows` 长度 = 24（`ACTIVE_WINDOW_SIZE`）；
  8. `avatarSeed == username`；
  9. `historicalTweets` 数量 5-10。
- 统计输出：总数/历史推文总数（人均）/职业/年龄段/文化背景/语言风格分布。
- 最小覆盖校验：每个职业 >= 8、每个年龄段 >= 40、每个文化背景 >= 30（PASS/FAIL）。
- 错误最多显示前 30 条，超出提示"还有 N 个错误未显示"。返回 0 全通过，1 有错误。
- 运行：

```bash
python tools/persona-gen/validate.py
```

## 图库系统

- 8 主题：`landscape`/`food`/`city`/`pet`/`sport`/`art`/`tech`/`nature`，每主题 25 张 SVG，共 200 张。
- `gallery/index.json`：按主题分组文件名列表。
- `GalleryImageTheme` 枚举与主题目录对应，`NONE` 表示无图。
- `LocalImageGalleryImpl.pickRandom(theme, accountId)`：
  1. `ensureIndexLoaded` 读 `index.json`（`Mutex` 保证一次）；
  2. 查 `recentlyUsedAssets(now-30天)` 排除已用；
  3. `candidateThemes` 指定主题置顶其余回退；
  4. 逐主题过滤 available 后 random；
  5. `recordUsage` 记录。
- `RoomImageUsagePort`：`imageHash` 字段直接用 `assetPath` 作唯一标识，30 天去重窗口进程重启仍有效。
- 资产路径如 `gallery/landscape/3.svg`，UI 层转 `file:///android_asset/gallery/landscape/3.svg` 加载。

## SVG 生成（tools/gallery-gen/generate.py）

- 纯 Python 标准库无外部依赖。`CANVAS_SIZE=1080`（1080x1080），`IMAGES_PER_THEME=25`。
- 输出 `app/src/main/assets/gallery/<theme>/<n>.svg` + `index.json`。单张目标 < 5KB（RISK-10 减小 APK 体积）。
- 8 主题配色（每主题 background 渐变起止色 + 6 色调色板）：landscape 绿 / food 橙 / city 灰蓝 / pet 棕黄 / sport 红 / art 紫 / tech 蓝 / nature 绿青。
- SVG 生成：`svg_header` 定义 `linearGradient` 背景；`gen_circle`/`gen_rect`/`gen_triangle` 随机几何图形（矩形可旋转，三角形 120 度间隔三顶点）；每张 3-7 个图形 `fill-opacity` 0.35-0.85；`gen_label` 左下角绘制主题名（英文）。
- 每主题固定种子 `hash(theme_name)&0xFFFFFFFF` 保证可复现。超 5KB 仅警告仍保留。
- `mix_color` 两种 hex 颜色按 ratio 线性混合；`fmt` 浮点格式化去尾零减小体积。
- 运行：

```bash
python tools/gallery-gen/generate.py
```

```python
# 主题配色与可复现种子

IMAGES_PER_THEME = 25

def seed_for_theme(theme_name: str) -> int:
    return hash(theme_name) & 0xFFFFFFFF
```

## 头像系统

- `avatars/index.txt`：220 行 UUID，顺序与 `personas/index.json` `accountIds` 一致。
- 头像为 SVG，由 `avatarSeed(username)` hash 映射到 gallery 8 类别之一，每类 25 张。
- `FeedUtils.avatarUriFromSeed` / `ProfileUtils.avatarUriFromSeed` 实现 hash 映射。
- Coil + `SvgDecoder.Factory` 解码 SVG（`feature-feed`/`feature-profile` 各有独立 `ImageLoader` Module）。

## 再生成流程

- 重新生成人设：

```bash
python tools/persona-gen/generate.py
```

- 校验：

```bash
python tools/persona-gen/validate.py
```

- 重新生成图库：

```bash
python tools/gallery-gen/generate.py
```

- 注意：生成后人设 `id` 会变，需同步更新 `avatars/index.txt`（`generate.py` 已自动写入）。
